### 1. Постановка задачи
#### Задание 1.
>Посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее.

#### Задание 2.
>Среди натуральных чисел, не превышающих 10^12, нужно найти количество чисел, соответствующих маске “*5?3*”, у которых сумма цифр делится на 7.

#### Задание 3.
>С клавиатуры вводится последовательность чисел, пока не будет введено отрицательное число. Нужно вывести на экран длину самой длинной подпоследовательности, >расположенной между двумя нулями. Если в последовательности меньше двух нулей, вывести 0.

#### Задание 4.
>Дан массив из n чисел. Нужно вывести количество пар элементов, для которых модуль разности не превышает заданное число d.
