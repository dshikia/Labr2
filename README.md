### 1. Постановка задачи
#### Задание 1.
>Посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее.

#### Задание 2.
>Среди натуральных чисел, не превышающих 10^12, нужно найти количество чисел, соответствующих маске “* 5?3 *”, у которых сумма цифр делится на 7.

#### Задание 3.
>С клавиатуры вводится последовательность чисел, пока не будет введено отрицательное число. Нужно вывести на экран длину самой длинной подпоследовательности, расположенной между двумя нулями. Если в последовательности меньше двух нулей, вывести 0.

#### Задание 4.
>Дан массив из n чисел. Нужно вывести количество пар элементов, для которых модуль разности не превышает заданное число d.

### 3. Выбор структуры данных

### 4. Алгоритм

#### Алгоритм выполнения программы:

#### Задание 1.
1. **Ввод данных:**  
   Программа считывает два целых числа, обозначенные как o и n.

2. **Расчет закономерности последовательности:**

   При o = 1, n = 10, получается следующая последовательность: 7, 49, 91, 70, 49, 91, 70, 49, 91, 70
   Можно заметить, что каждое следующее число получается путем сложения цифр предыдущего числа и умножения их суммы на 7.
   То есть 49 = (7+0)*7, 91 = (4+9)*7 и т.д.

4. **Вывод результата:**
   На экран выводится строка, соответствующая определенной последовательности.

#### Задание 2.
1. **Ввод данных:**
   Используем заданную в условии маска.
  
3. **Суть программы:**
   В условии задан диапазон 10^12, но так как программа должна перебрать огромное количество чисел, уменьшим его до 10^9.
   В первом цикле перебираем числа до 10^9, а затем преобразовываем их в строку.
   Во втором цикле проверяем позиции в строке числа.
   Вводим переменную num - копия числа для дальнейшего вычисления суммы.

5. **Вывод результата:**
    На экран выводится число, соответствующее количеству чисел, соответствующих маске “* 5?3 *”, у которых сумма цифр делится на 7.

#### Задание 3.
1. **Ввод данных:**  
   Программа считывает целое число n, до тех пор, пока n≥0.

2. **Суть программы:**  
   С каждым новым значением n увеличивается индекс, соответсвующий номеру элемента в последовательности (изначально ind = 0).
   При вводе 0 его индекс фиксируется.
   При повторном вводе 0 считается длина последовательности, заключенной между нулями.
   Если введен один ноль или не введен ни один, то значение переменной длины не изменяется и остается равной нулю.

4. **Вывод результата:**
   На экран выводится число, соответствующе значению длины самой длинной подпоследовательности, расположенной между двумя нулями. Либо выводится 0, если в последовательности меньше двух 0.

#### Задание 4.
1. **Ввод данных:**  
   Программа считывает два целых числа, обозначенные как n и d. Далее вводятся сами элементы массива.
   

3. **Суть программы:**  
   В программе присутвует два цикла. Первый перебирает элементы массива от i = 0, а второй - от i+1. Так получается пара чисел массива.
   Далее находим модуль разности этих элементов. Если разности не превышает заданное число d, то увеличиваем счетчик, отвечающий за количество пар. 

5. **Вывод результата:**
   На экран выводится число, соответствующее количеству пар элементов, для которых модуль разности не превышает заданное число d.


### 5. Программа
#### Задание 1.
```import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        int o = in.nextInt();
        int n = in.nextInt();
        int l = n+1;
        int[] a = new int[l];
        int s;

        a[0] = o;

        for (int i = 1; i < l; i++) {
            s=0;
            int pr = a[i - 1];
            while (pr > 0) {
                int x = pr % 10;
                s += x;
                pr = pr / 10;

            }
            a[i] = s * 7;
        }
        for (int i = 1; i < l; i++)
            out.print(a[i] + " ");
    }
}
```

#### Задание 2.
```import java.io.IOException;
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) throws IOException {
        long k = 0;
        String s = "";
        for (long ch = 1; ch <= (long)(Math.pow(10,9)); ch++) {
            s = Long.toString(ch);

            boolean flag = false;

            for (int i = 0; i < s.length(); i++) {
                if (s.charAt(i) == '5' && i + 2 < s.length()) {
                    if (s.charAt(i + 2) == '3') {
                        flag = true;
                        break;
                    }
                }
            }

            if (flag==false)
                continue;

            long num = ch;
            int sum = 0;
            while (num > 0) {
                sum += num % 10;
                num /= 10;
            }

            if (sum % 7 == 0) {
                k++;
            }
        }

        out.println(k);
    }
}
```

#### Задание 4.
```import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args) {
        int n = in.nextInt();
        int d =in.nextInt();
        int[] a = new int[n];

        for (int i = 0; i<n; i++)
            a[i] = in.nextInt();

        int k = 0;
        for (int i = 0; i<n; i++){
            for (int j = i+1; j<n; j++){
                if (Math.abs(a[i]-a[j]) <= d)
                    k++;
            }
        }

        out.print(k);
    }
}
```
